About Cephalometric Landmark

we proposed the landmark detection system for dental image analysis by constructing multiple CNN-based regression systems predicting individual coordinate values of landmarks, independently. In experiments, the proposed system showed promising performance by successfully locating the landmarks with considerable margins. Use of deeper networks with larger input images would enhance the power of the proposed system. As far as we know, this was the first attempt to construct the end-to-end learning framework for cephalometric landmark detection task.

Results

For this project i have build CNN-based regression system for predicting all the 19 co-ordinates with a minimal error.In the proposed model we have five convolution layers,5 max-polling layers, 2 dropout layers and one fully connected layer.We use input images of size 64 × 64 which is scaled from the original input images. The first convolutional layer consists of six feature maps with a 5 × 5 kernel, and is followed by 2 × 2 max pooling to reduce the size of feature maps. The second convolutional layer consists of twelve feature maps, with a 5 × 5 kernel, and is also followed by 2 × 2 max pooling to subsample the feature maps,The third convolutional layer consists of sixty-four feature maps, with a 2 × 2 kernel, and is also followed by 2 × 2 max pooling to subsample the feature maps and the same layer is repeated on more time and one more convolution layer with 128 feature maps. Finally, a fully-connected layer computes binary-sized output vector consisting of probabilities in which the first probability value corresponds to the normalized coordinate variable of landmarks. The proposed CNN regression system is trained on training images and corresponding coordinate variables of landmarks.
